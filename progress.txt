## Codebase Patterns

- CLI commands live in `cli/src/commands/<name>.ts` and are registered in `cli/src/index.ts`
- Command option types are defined in `cli/src/types.ts` and imported into command files
- The `exec()` utility in `cli/src/utils/exec.ts` handles spawning AI tools; pass `stdin` for non-copilot tools, `autoApprove` for copilot
- Templates (DEVELOPER.md, PLANNER.md, DOCTOR.md) live in `cli/templates/` and are copied by `ralph init`; `getPackageDir()` from `utils/files.ts` resolves the template dir at runtime
- `validatePrdFile()` in `cli/src/utils/prd.ts` returns `{ valid, errors[] }` ‚Äî use this before any AI tool invocation
- Build: `cd cli && npm run build` (tsup, ESM only); typecheck: `npm run typecheck`
