## Codebase Patterns
- ralph-cli is a TypeScript ESM CLI. Run `npm run typecheck` in cli/ for type checking, `npm run build` then `node dist/index.js` for validation.
- prd.json schema is validated by `cli/src/utils/prd.ts`. To add new required fields to userStories, add a `require*` call in the `userStories.forEach` block and add a helper if needed.
- The root prd.json, prd.json.example, and cli/templates/prd.json.example must all stay in sync when updating the user story schema shape.
