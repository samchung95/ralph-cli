## Codebase Patterns
- `getPackageDir()` in `cli/src/utils/files.ts` returns the Ralph package root (one level up from `dist/index.js`). Use `join(getPackageDir(), "templates")` to locate bundled templates at runtime.
- Runtime prompt cleanup pattern: before writing a tool's runtime prompt file, read and store its existing content (or null if absent). In a `finally` block, restore the previous content or delete the file if Ralph created it.
- `removePathIfExists` is already available from `cli/src/utils/files.ts`; use it for safe file deletion.
- For safe stale-artifact cleanup, compare normalized content hashes against known bundled template/runtime hashes instead of deleting by filename alone. This preserves user-customized files that happen to share Ralph artifact names.
